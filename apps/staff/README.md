# スタッフ向けシステム管理アプリ

システム管理者向けのアプリケーションです。テナント管理、プラン管理、システム設定を行います。

## 機能概要

システム全体を管理する管理者向けの機能を提供します。

## 技術スタック

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: JWT (localstorage)

## セットアップ

### 開発サーバー起動

```bash
# スタッフアプリのみ起動
pnpm -F staff dev

# または全アプリ起動
pnpm dev
```

アプリは http://localhost:3001 で起動します。
